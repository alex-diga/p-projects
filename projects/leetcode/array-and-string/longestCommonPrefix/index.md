# 练习

[最长公共前缀](https://leetcode.cn/leetbook/read/array-and-string/ceda1/)

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串  ""。

## 示例 1

```text
输入：strs = ["flower","flow","flight"]
输出："fl"
```

## 示例 2

```text
输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。
```

## 思路

1. 纵向扫描：遍历第一个字符串的字符，并与其余字符串相应位置的字符比较
